# Skills

Protocol-level skills that enforce ai-context rules.

- NOT memory systems
- Do NOT store project knowledge
- Only bootstrap and validate protocol compliance

Available:
- `ctx` — quick context check (list decisions, show latest, report status)
- `ai-context-init` — full initialization (create .ai-context/, configure CLAUDE.md/AGENTS.md)
- `ai-context-lint` — validate protocol and project-local context structure
